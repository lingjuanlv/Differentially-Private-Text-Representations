# Differentially-Private-Text-Representations
Code for SIGIR'20 paper "Towards Differentially Private Text Representations".

# How to run:
```
Data folder: data/sentiment_analysis
python sentiment_analysis_ldp_github.py
```

# Requirements:
```
python3
tensorflow
keras
```
